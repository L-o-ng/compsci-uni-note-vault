#notes 

If we have a function $f:V \to W$ where $V$ and $W$ are vector spaces over a field $F$ then $f$ is a linear map if for any vectors $\mathbf{u},\mathbf{v} \in V\text{ and }a \in F$.
$$\array{f(\mathbf{u},\mathbf{v})=f(\mathbf{u})+f(\mathbf{v})\\f(a\mathbf{v})=af(\mathbf{v})}$$
This means than a linear map *respects* linear combinations.
Linear maps are sometimes called *linear* *transformations* or *vector space homomorphisms*.
If $V=W$ then $f$ is an *endomorphism*.
All linear maps preserve lines.
### Examples
+ Scaling on $\mathbb{R}^3$
+ Reflection
+ Rotation
+ [[Shearing]]
+ [[Projection]]
+ Differentiation on polynomials
