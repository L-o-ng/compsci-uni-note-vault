Let $f(x)$ be a function defined on an interval $[a,b]$.
A point $x_{0}\in[a,b]$ is:
1) An absolute maximum if $f(x_{0})\ge f(x)\forall x \in[a,b]$
2) An absolute minimum if $f(x_{0})\le f(x)\forall x \in[a,b]$
3) A local maximum if $\exists \updelta > 0:f(x_{0})\ge f(x_{0}+h)\forall |h|<\updelta$
4) A local minimum if $\exists \updelta > 0:f(x_{0})\le f(x_{0}+h)\forall |h|<\updelta$
