We can also use a **change of variables** to simplify the limit:
$$\array{\lim_{ x \to a } f(x)=L\text{ does not depend on }x\text{!}\\ \text{So we can change the variables.}}$$
## Example
### Example 1
$$\Large\begin{align}
\lim_{ x \to 0 } \frac{\sin3x}{x}&:\text{Let }y=3x, \text{ so as }x \to 0, y \to 0, \\
\text{Then }&=\lim_{ y \to 0 } \frac{\sin y}{\frac{y}{3}} \\
&=\lim_{ y \to 0 } 3 \frac{\sin y}{y} \\
&=\underline{3}
\end{align}$$
### Example 2
$$\Large \array{\begin{align}
\lim_{ x \to 1 } \frac{\sin(3x-3)}{x-1}&=\lim_{ y \to 0 } \frac{\sin y}{\frac{y}{3}} \\
&=\underline{3} \\
\end{align}\\ 
\text{By letting }y=3x-3\text{ so as }x \to 1, y \to 3-3=0
} $$
