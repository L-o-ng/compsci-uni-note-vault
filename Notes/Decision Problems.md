#linker 

**Decision** **Problems** share a common feature:
+ They consist of a set of instances such that, corresponding to any instance is the answer yes or no.
## Formal Definition
>A decision problem $D$ consists of:
>	A set of instances $/$
>	A subset $Y \subseteq /$ of yes-instances.
>	And a set of no-instances, $/ \setminus Y$
