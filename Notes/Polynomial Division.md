For any polynomial $p(z)$ and $a \in \mathbb{C}$, we can write: $$p(z)=(z-a)q(z)+c$$
## Consequence
So, if $w$ is a root of $p(z)$ ie $p(w)=0$
Then: $$p(z)=(z-w)q(z)+c, \text{but }c=0\text{ here}$$
$$\text{So, }p(z)=(z\cdot w)q(z) \text{ if } w\text{ is a root of }p(z)$$
