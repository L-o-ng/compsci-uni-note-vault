#definition 

Consider the vector $\left(\array{x_{1}\\ x_{2}}\right)$.
+ How long is it?
$$\eqalign{\text{By pythagoras, } & l^2=x_{1}^2+x_{2}^2\\ \text{In 3D, } & l^2=x_{1}^2+x_{2}^2+x_{3}^2}$$
+ So, if $x=(x_{1},\dots,x_{n}) \in \mathbb{R}$ then the *Euclidean Norm* is defined as: $$\Vert\mathbf{x}\Vert_{2}=\sqrt{ \sum_{i}x_{i}^2 }$$often shortened to $\Vert\mathbf{x}\Vert$.
This norm is treated as the 'default' norm.
