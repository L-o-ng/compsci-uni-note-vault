#notes 

Language design has continuously evolved, driven by the following factors: 
+ **Productivity**
	+ C, C++, etc are detailed and lead to lots of lines.
	+ Rapid Application Development languages like Ruby and Python were created as a result.
+ **Reliability**
	+ Features are introduced to enhance error detection and type checking.
+ **Security**
+ **Execution Speed**
	+ Imperative languages are tied to sequential computers.
+ **Curiosity and Style**
