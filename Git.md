## Key Concepts
+ A **file** in a path.
+ A **commit** - a snapshot of a collection of files at a particular time.
+ A **branch** - a linear sequence of commits.
+ A **repository** - a collection of many branches of a project.
+ A **remote** - another place where a repository is stored.

## Key Commands in git
```
git init // initialises repo
git add // stages file for commit
git status //check staged files
git commit // commit staged files to branch, opens editor
git push // commit to remote

git commit -m // commits and avoids editor
git log // shows change history
git remote add origin <link> // adds remote repository
git push origin main // pushes branch main to remote origin
```
